# getWPCategories 插件 #
因业务需求，需要获取 WP 中的文章分类目录，因此编写实现当前插件。功能如下：

1. 实现手动生成 wp 已经发布文章的固定连接的 json 文件。
2. 实现手动生成 wp 所有已经发布文章的分类目录 json 文件。
3. 实现用户使用 post 方式请求，发送 wp post ID 返回固定链接。
4. 实现用户使用 post 方式请求，发送 wp post ID 返回分类目录。
